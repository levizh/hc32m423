功能描述
================================================================================
本样例主要展示TIMER0的基本计数功能，在中断程序中翻转RGB的绿灯。

说明
使能TIMER0 计数匹配中断，异步计数方式，LRC作为计数时钟源，频率32.768KHZ,
中断频率为0.5S一次;
同时使能INTP1（SW2）下降沿产生内部触发事件，触发事件可以使定时器停止计数。


================================================================================

无

---------------------
无

================================================================================
使用步骤
================================================================================
1） 打开工程编译并全速运行。
2） RGB绿灯以1Hz的频率闪烁。
3） 短按按键SW2，产生内部触发事件，此时定时器计数停止，绿灯停止闪烁。

================================================================================
注意
================================================================================
同步计数方式下时钟源可选HCLK、内部硬件触发事件
异步计数方式下时钟源只能是LRC
