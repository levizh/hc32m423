平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。


功能描述
================================================================================
本样例为flash连续编程，全擦除。

================================================================================
使用步骤
================================================================================
1）重新编译；
2）启动IDE的下载和调试功能；
3）全速运行，可观察到绿灯亮。
若出现红灯亮,则断点查看具体是扇区擦除，连续编程还是全擦除出错。

说明：
此样例需运行在RAM演示，IAR无需额外配置；
KEIL环境的RAM工程参考设置https://blog.csdn.net/wjs1033/article/details/51745494
